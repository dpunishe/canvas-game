# canvas-game
simple html(canvas) game with JavaScript
<br>
controls for player 1(left) [A] [W] [D] for move and [space] for attack
<br>
controls for player 2(right) [ArrowLeft] [ArrowUp] [ArrowRight] for move and [ArrowDown] for attack
<br>
